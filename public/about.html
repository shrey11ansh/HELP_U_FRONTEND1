ABOUT US

ALWAYS THERE TO HELP

This “HELP U” project is developed using specific programming languages. The main aim of this project is to develop an online website which cover the details of college like event details, placement details, academic details, complaint portal, photo gallery, etc. This website is primarily dedicated to college students. All the records are stored in MySQL Database. The proposed software will also reduce the cumbersome paperwork, manual labour as well as communication cost.
